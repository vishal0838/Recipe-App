# 🍳 Recipe Finder App

A sleek, responsive web application that allows users to search for recipes, view ingredients, and get cooking instructions in real-time. Built with a focus on clean UI and efficient API integration.


## ✨ Features
- **Real-time Search:** Instantly find recipes by keyword using the [Mention API Name, e.g., MealDB] API.
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens using CSS Flexbox/Grid.
- **Dynamic UI:** Content is rendered dynamically using Vanilla JavaScript (no frameworks).
- **Detailed View:** Click on any recipe to see a full list of ingredients and step-by-step instructions.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Custom styling with a focus on modern UI/UX.
- **JavaScript (ES6+):** - Fetch API for asynchronous data handling.
  - DOM Manipulation for updating the interface.
  - Event Listeners for user interactivity.

## 📖 What I Learned
During the development of this project, I strengthened my understanding of:
1. **Asynchronous Programming:** Handling `async/await` and `.then()` to manage API responses and potential errors.
2. **State Management:** Displaying "Loading..." spinners or "No Results Found" messages to improve user experience.
3. **CSS Layouts:** Mastering responsive images and grid layouts without relying on external libraries like Bootstrap.
